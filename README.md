## Matrix-new
##### 遵循开源协议
 MIT
 ##### 项目简介
 `此项目是基于Windows平台EasyX图形库开发的迷宫游戏,具有多种自动生成迷宫地图特性以及迷宫寻径功能`
 ##### 开发工具及运行环境
 ###### 开发IDE环境 :  Visual Studio 2019
 ###### 代码管理工具:  Git
 ###### 开发语言:C++
 ######  程序运行环境(开发环境为(Windows10)其他兼容性未知) 
 ##### 依赖库
 EasyX 图形界面库
 EasyX官网：[EasyX](https://easyx.cn/)
