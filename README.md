# maze

一个简单的C++编写的windows迷宫游戏，支持迷宫随机生成、迷宫寻路径等功能  

## 快速开始

获取项目  

```shell
git clone https://hub.fastgit.xyz/gaowanlu/maze.git  
```

使用 Visual Studio 打开项目 ，并进行下一步获取EasyX  

## 环境搭建

### 1、EasyX官网：[EasyX](https://easyx.cn/)
